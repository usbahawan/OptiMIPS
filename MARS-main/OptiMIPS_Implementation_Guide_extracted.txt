1.  Originality Verification
Before building, we verified no identical tool exists. Research findings:
The only pipeline plugin published for MARS is PBSE (Pipelined Simulator and Branch Explainer, 2019 IEEE). It simulates pipeline execution and shows hazards — but does NOT reorder instructions automatically.
MIPS X-Ray (2014) shows the datapath and detects hazards visually — no optimization.
The PBSE paper's own Future Work section lists automatic instruction reordering as an open problem. Your project implements exactly that.
No GitHub project, thesis, or IEEE/ACM paper describes an instruction-reordering optimizer plugin for MARS.
Conclusion: OptiMIPS is original. It fills the gap explicitly identified in published literature.
2.  What OptiMIPS Does
OptiMIPS is a MARS plugin. The user writes a MIPS program in MARS as normal. When they click Optimize, the plugin:
Reads the assembled instruction list from MARS's internal memory.
Builds a Data Dependency Graph (DDG) — a directed graph where an edge from A→B means B must stay after A.
Runs a list scheduling algorithm to reorder instructions, filling stall slots with safe independent instructions.
Simulates both the original and optimized sequences through a simple pipeline model, counting stall cycles.
Displays original code vs optimized code side-by-side, with colored highlights and a CPI comparison panel.
What makes it novel over just visualizing hazards: the tool actively eliminates the hazards by reordering, then proves the improvement with measured CPI numbers. No existing MARS tool does this.

4.  Member 1 — Dependency Graph Builder
4.1  What to build
A class called DependencyGraph that takes an ArrayList of MipsInstruction objects and outputs a directed graph. An edge from node i to node j means: instruction j depends on instruction i and must stay after it in the schedule.

4.3  The DependencyGraph class
This is the core algorithm. It checks every pair of instructions for all three hazard types:
DependencyGraph.java — complete implementation
package mars.tools;
import java.util.*;
public class DependencyGraph {
    // adjacency list: edges[i] = list of instruction indices that
    // MUST come after instruction i
    private List<Set<Integer>> edges;
    private int n;
    public DependencyGraph(List<MipsInstruction> instrs) {
        n = instrs.size();
        edges = new ArrayList<>();
        for (int i = 0; i < n; i++) edges.add(new HashSet<>());
        for (int i = 0; i < n; i++) {
            for (int j = i + 1; j < n; j++) {
                if (hasEdge(instrs.get(i), instrs.get(j),
                            instrs, i, j)) {
                    edges.get(i).add(j); // j must stay after i
                }
            }
        }
    }
    private boolean hasEdge(MipsInstruction a, MipsInstruction b,
                             List<MipsInstruction> all, int ai, int bi) {
        // RULE 1: Nothing crosses a branch/jump
        // If ANY instruction between a and b (inclusive) is a branch,
        // lock everything after it.
        for (int k = ai; k < bi; k++) {
            if (all.get(k).isBranch) return true;
        }
        // RULE 2: Memory instructions never reorder with each other
        if ((a.isMemLoad || a.isMemStore) &&
            (b.isMemLoad || b.isMemStore)) return true;
        // RULE 3a: RAW — a writes a register that b reads
        if (a.dest != -1) {
            for (int src : b.sources) {
                if (src == a.dest) return true;
            }
        }
        // RULE 3b: WAR — a reads a register that b writes
        if (b.dest != -1) {
            for (int src : a.sources) {
                if (src == b.dest) return true;
            }
        }
        // RULE 3c: WAW — both write the same register
        if (a.dest != -1 && b.dest != -1 && a.dest == b.dest)
            return true;
        return false;
    }
    // Returns true if instruction j must stay after instruction i
    public boolean mustFollow(int i, int j) {
        return edges.get(i).contains(j);
    }
    // Returns all predecessors of instruction j
    // (instructions that must come before j)
    public Set<Integer> getPredecessors(int j) {
        Set<Integer> preds = new HashSet<>();
        for (int i = 0; i < n; i++) {
            if (edges.get(i).contains(j)) preds.add(i);
        }
        return preds;
    }
    // Returns the dependency type as a String (for UI display)
    public String getHazardType(MipsInstruction a, MipsInstruction b) {
        if (a.dest != -1) {
            for (int src : b.sources)
                if (src == a.dest) return "RAW";
        }
        if (b.dest != -1) {
            for (int src : a.sources)
                if (src == b.dest) return "WAR";
        }
        if (a.dest != -1 && b.dest != -1 && a.dest == b.dest)
            return "WAW";
        if (a.isBranch || b.isBranch) return "CTRL";
        if ((a.isMemLoad||a.isMemStore) &&
            (b.isMemLoad||b.isMemStore)) return "MEM";
        return "NONE";
    }
}

5.  Member 2 — List Scheduler (the Optimizer)
5.1  What list scheduling does
List scheduling scans the instruction list from top to bottom. Whenever it detects a stall slot — a gap where the pipeline would have to wait — it looks forward for an instruction that:
Has all its predecessors (in the dependency graph) already placed above it.
Does not introduce a new hazard with the instruction currently in the pipeline.
If such an instruction exists, it moves it up to fill the stall. The result is the same program logic, fewer wasted cycles.
5.2  Complete Scheduler implementation
Scheduler.java — complete implementation
package mars.tools;
import java.util.*;
public class Scheduler {
    // latency[mnemonic] = cycles the instruction takes in EX stage
    // lw has latency 2 (load-use hazard needs 1 extra stall if not filled)
    private static final Map<String,Integer> LATENCY = new HashMap<>();
    static {
        LATENCY.put("lw",  2);  LATENCY.put("lb",  2);  LATENCY.put("lh", 2);
        LATENCY.put("add", 1);  LATENCY.put("sub", 1);  LATENCY.put("and",1);
        LATENCY.put("or",  1);  LATENCY.put("mul", 1);  LATENCY.put("div",1);
        LATENCY.put("addi",1);  LATENCY.put("slt", 1);  LATENCY.put("sw", 1);
    }
    private List<MipsInstruction> original;
    private DependencyGraph       graph;
    public Scheduler(List<MipsInstruction> instrs, DependencyGraph graph) {
        this.original = instrs;
        this.graph    = graph;
    }
    // Returns the reordered instruction list
    public List<MipsInstruction> schedule() {
        int n = original.size();
        boolean[] scheduled = new boolean[n]; // already placed?
        List<MipsInstruction> result = new ArrayList<>();
        while (result.size() < n) {
            // Find the next instruction that is ready to place:
            // all its predecessors are already in result
            int best = -1;
            for (int i = 0; i < n; i++) {
                if (scheduled[i]) continue;
                if (allPredecessorsScheduled(i, scheduled)) {
                    // Prefer instructions that fill a stall slot
                    if (best == -1) {
                        best = i;
                    } else {
                        // Heuristic: prefer the instruction with the
                        // most successors (critical path first)
                        if (successorCount(i) > successorCount(best))
                            best = i;
                    }
                }
            }
            if (best == -1) break; // safety — should not happen
            scheduled[best] = true;
            result.add(original.get(best));
        }
        return result;
    }
    private boolean allPredecessorsScheduled(int j, boolean[] scheduled) {
        for (int i = 0; i < original.size(); i++) {
            if (i == j) continue;
            if (graph.mustFollow(i, j) && !scheduled[i])
                return false;
        }
        return true;
    }
    private int successorCount(int i) {
        int count = 0;
        for (int j = 0; j < original.size(); j++) {
            if (graph.mustFollow(i, j)) count++;
        }
        return count;
    }
    // Count stall cycles for a given instruction sequence
    public static int countStalls(List<MipsInstruction> instrs) {
        int stalls = 0;
        // pipeline: track when each register becomes ready
        int[] ready = new int[32]; // ready[r] = cycle when reg r is ready
        Arrays.fill(ready, 0);
        int cycle = 0;
        for (MipsInstruction instr : instrs) {
            cycle++;
            // Check if any source register is not yet ready
            int waitUntil = cycle;
            for (int src : instr.sources) {
                if (src >= 0 && src < 32)
                    waitUntil = Math.max(waitUntil, ready[src]);
            }
            int stalledCycles = waitUntil - cycle;
            stalls += stalledCycles;
            cycle = waitUntil;
            // Mark when destination register will be ready
            if (instr.dest >= 0) {
                int lat = LATENCY.getOrDefault(instr.mnemonic, 1);
                ready[instr.dest] = cycle + lat;
            }
        }
        return stalls;
    }
    // Calculate CPI given stall count and instruction count
    public static double calcCPI(int instrCount, int stalls) {
        if (instrCount == 0) return 0;
        return (double)(instrCount + stalls) / instrCount;
    }
}
6.  Member 3 — MARS Integration & Parser
6.1  Converting ProgramStatement to MipsInstruction
MARS gives you ProgramStatement objects. You need to convert these into your MipsInstruction format. Here is the complete parser:
MarsParser.java — complete implementation
package mars.tools;
import mars.Globals;
import mars.ProgramStatement;
import java.util.*;
public class MarsParser {
    // R-type instructions that write dest=op[0], read src=op[1],op[2]
    private static final Set<String> R_TYPE = new HashSet<>(Arrays.asList(
        "add","sub","and","or","xor","nor","slt","sltu",
        "mul","div","sll","srl","sra"
    ));
    // I-type: dest=op[0], src=op[1]  (addi, andi, ori, slti)
    private static final Set<String> I_TYPE = new HashSet<>(Arrays.asList(
        "addi","addiu","andi","ori","xori","slti","sltiu"
    ));
    // Load: dest=op[0], base=op[2]  (lw $t0, offset($t1))
    private static final Set<String> LOADS = new HashSet<>(Arrays.asList(
        "lw","lb","lbu","lh","lhu"
    ));
    // Store: src=op[0], base=op[2]  (sw $t0, offset($t1))
    private static final Set<String> STORES = new HashSet<>(Arrays.asList(
        "sw","sb","sh"
    ));
    // Branch/Jump: no dest register
    private static final Set<String> BRANCHES = new HashSet<>(Arrays.asList(
        "beq","bne","blez","bgtz","bltz","bgez",
        "j","jal","jr","jalr"
    ));
    public List<MipsInstruction> parse() {
        List<MipsInstruction> result = new ArrayList<>();
        try {
            ArrayList<ProgramStatement> stmts =
                Globals.program.getMachineList();
            int idx = 0;
            for (ProgramStatement stmt : stmts) {
                String mn = stmt.getInstruction().getName().toLowerCase();
                int[]  op = stmt.getOperands(); // raw operand array
                String txt = stmt.getSource();  // original asm text
                int    dest    = -1;
                int[]  sources = new int[0];
                boolean isLoad  = LOADS.contains(mn);
                boolean isStore = STORES.contains(mn);
                boolean isBr    = BRANCHES.contains(mn);
                if (R_TYPE.contains(mn) && op.length >= 3) {
                    dest    = op[0];
                    sources = new int[]{op[1], op[2]};
                } else if (I_TYPE.contains(mn) && op.length >= 2) {
                    dest    = op[0];
                    sources = new int[]{op[1]};
                } else if (isLoad && op.length >= 3) {
                    dest    = op[0];
                    sources = new int[]{op[2]}; // base register
                } else if (isStore && op.length >= 3) {
                    dest    = -1;
                    sources = new int[]{op[0], op[2]}; // value + base
                } else if (isBr && op.length >= 2) {
                    dest    = -1;
                    sources = new int[]{op[0], op[1]};
                }
                result.add(new MipsInstruction(
                    idx++, mn, dest, sources,
                    isLoad, isStore, isBr, txt));
            }
        } catch (Exception e) {
            e.printStackTrace();
        }
        return result;
    }
}
7.  Member 4 — Main Plugin Class & UI
This is the class MARS finds and adds to the Tools menu. It wires together all other classes and builds the Swing UI.
OptiMIPS.java — complete main plugin class
package mars.tools;
import mars.mips.hardware.*;
import javax.swing.*;
import javax.swing.table.*;
import java.awt.*;
import java.awt.event.*;
import java.util.*;
public class OptiMIPS extends AbstractMarsToolAndApplication {
    // UI components
    private JTable  origTable, optTable;
    private JLabel  origCpiLabel, optCpiLabel, gainLabel;
    private JTextArea hazardLog;
    private JButton optimizeBtn;
    public OptiMIPS() {
        super("OptiMIPS — Pipeline Optimizer", "OptiMIPS");
    }
    public String getName() { return "OptiMIPS — Pipeline Optimizer"; }
    @Override
    protected JComponent buildMainDisplayArea() {
        JPanel root = new JPanel(new BorderLayout(8,8));
        root.setBorder(BorderFactory.createEmptyBorder(10,10,10,10));
        // ── Top toolbar ──
        optimizeBtn = new JButton("▶  Optimize Program");
        optimizeBtn.setFont(new Font("Arial", Font.BOLD, 13));
        optimizeBtn.setBackground(new Color(46, 95, 163));
        optimizeBtn.setForeground(Color.WHITE);
        optimizeBtn.addActionListener(e -> runOptimizer());
        JPanel toolbar = new JPanel(new FlowLayout(FlowLayout.LEFT));
        toolbar.add(optimizeBtn);
        root.add(toolbar, BorderLayout.NORTH);
        // ── Centre: two side-by-side code tables ──
        origTable = makeTable();
        optTable  = makeTable();
        JPanel codePanel = new JPanel(new GridLayout(1,2,10,0));
        codePanel.add(wrapTable(origTable, "Original Code",
                      new Color(255,235,235)));
        codePanel.add(wrapTable(optTable,  "Optimized Code",
                      new Color(232,245,232)));
        root.add(codePanel, BorderLayout.CENTER);
        // ── Bottom: stats + hazard log ──
        origCpiLabel = new JLabel("CPI: —");
        optCpiLabel  = new JLabel("CPI: —");
        gainLabel    = new JLabel("");
        gainLabel.setFont(new Font("Arial", Font.BOLD, 14));
        gainLabel.setForeground(new Color(14, 100, 70));
        hazardLog = new JTextArea(5, 40);
        hazardLog.setFont(new Font("Courier New", Font.PLAIN, 12));
        hazardLog.setEditable(false);
        JPanel statsRow = new JPanel(new GridLayout(1,3,10,0));
        statsRow.add(makeStatPanel("Before", origCpiLabel,
                     new Color(254,226,226)));
        statsRow.add(makeStatPanel("After",  optCpiLabel,
                     new Color(220,252,231)));
        statsRow.add(makeStatPanel("Gain",   gainLabel,
                     new Color(237,233,254)));
        JPanel bottom = new JPanel(new BorderLayout(0,6));
        bottom.add(statsRow, BorderLayout.NORTH);
        bottom.add(new JScrollPane(hazardLog), BorderLayout.CENTER);
        root.add(bottom, BorderLayout.SOUTH);
        return root;
    }
    // ── Core: runs when user clicks Optimize ──
    private void runOptimizer() {
        hazardLog.setText("");
        // 1. Parse instructions from MARS
        MarsParser parser = new MarsParser();
        List<MipsInstruction> original = parser.parse();
        if (original.isEmpty()) {
            hazardLog.setText("No assembled program found. "+
                              "Assemble your MIPS code first.");
            return;
        }
        // 2. Build dependency graph
        DependencyGraph graph = new DependencyGraph(original);
        // 3. Schedule (reorder) instructions
        Scheduler sched   = new Scheduler(original, graph);
        List<MipsInstruction> optimized = sched.schedule();
        // 4. Count stalls and calculate CPI
        int stallsBefore = Scheduler.countStalls(original);
        int stallsAfter  = Scheduler.countStalls(optimized);
        double cpiBefore = Scheduler.calcCPI(original.size(), stallsBefore);
        double cpiAfter  = Scheduler.calcCPI(optimized.size(), stallsAfter);
        double gain = ((cpiBefore - cpiAfter) / cpiBefore) * 100.0;
        // 5. Update UI
        populateTable(origTable, original, optimized, graph, true);
        populateTable(optTable,  optimized, original, graph, false);
        origCpiLabel.setText(String.format("CPI: %.3f  (%d stalls)",
                             cpiBefore, stallsBefore));
        optCpiLabel.setText(String.format("CPI: %.3f  (%d stalls)",
                             cpiAfter, stallsAfter));
        if (gain > 0.1)
            gainLabel.setText(String.format("+%.1f%% improvement", gain));
        else
            gainLabel.setText("No improvement possible");
        // 6. Log hazard explanations
        logHazards(original, graph);
    }
    private void logHazards(List<MipsInstruction> instrs,
                             DependencyGraph graph) {
        StringBuilder sb = new StringBuilder();
        sb.append("Hazard analysis:\n");
        for (int i = 0; i < instrs.size(); i++) {
            for (int j = i+1; j < instrs.size(); j++) {
                if (graph.mustFollow(i, j)) {
                    String type = graph.getHazardType(
                        instrs.get(i), instrs.get(j));
                    sb.append(String.format(
                        "  Line %d → Line %d : %s  [%s → %s]%n",
                        i+1, j+1, type,
                        instrs.get(i).originalText.trim(),
                        instrs.get(j).originalText.trim()));
                }
            }
        }
        hazardLog.setText(sb.toString());
    }
    // ── UI helpers ──
    private JTable makeTable() {
        DefaultTableModel model = new DefaultTableModel(
            new String[]{"#", "Instruction", "Hazard"}, 0) {
            public boolean isCellEditable(int r, int c) { return false; }
        };
        JTable t = new JTable(model);
        t.setFont(new Font("Courier New", Font.PLAIN, 12));
        t.setRowHeight(22);
        t.getColumnModel().getColumn(0).setMaxWidth(36);
        t.getColumnModel().getColumn(2).setMaxWidth(60);
        return t;
    }
    private JPanel wrapTable(JTable t, String title, Color headerColor) {
        JPanel p = new JPanel(new BorderLayout());
        JLabel lbl = new JLabel(" " + title);
        lbl.setFont(new Font("Arial", Font.BOLD, 13));
        lbl.setOpaque(true);
        lbl.setBackground(headerColor);
        lbl.setBorder(BorderFactory.createEmptyBorder(4,6,4,6));
        p.add(lbl, BorderLayout.NORTH);
        p.add(new JScrollPane(t), BorderLayout.CENTER);
        p.setBorder(BorderFactory.createLineBorder(
            new Color(200,200,200)));
        return p;
    }
    private JPanel makeStatPanel(String title, JLabel lbl, Color bg) {
        JPanel p = new JPanel(new BorderLayout());
        p.setBackground(bg);
        p.setBorder(BorderFactory.createCompoundBorder(
            BorderFactory.createLineBorder(new Color(200,200,200)),
            BorderFactory.createEmptyBorder(8,10,8,10)));
        JLabel t = new JLabel(title);
        t.setFont(new Font("Arial", Font.BOLD, 12));
        lbl.setFont(new Font("Arial", Font.PLAIN, 13));
        p.add(t, BorderLayout.NORTH);
        p.add(lbl, BorderLayout.CENTER);
        return p;
    }
    private void populateTable(JTable tbl,
                                List<MipsInstruction> instrs,
                                List<MipsInstruction> other,
                                DependencyGraph graph,
                                boolean isOriginal) {
        DefaultTableModel m = (DefaultTableModel) tbl.getModel();
        m.setRowCount(0);
        for (int i = 0; i < instrs.size(); i++) {
            MipsInstruction instr = instrs.get(i);
            // Check if this instruction has any dependency
            boolean hasHazard = false;
            String  hazType   = "";
            if (isOriginal) {
                for (int j = i+1; j < instrs.size(); j++) {
                    if (graph.mustFollow(instr.index, instrs.get(j).index)) {
                        hazType   = graph.getHazardType(instr, instrs.get(j));
                        hasHazard = true;
                        break;
                    }
                }
            }
            m.addRow(new Object[]{
                i+1,
                instr.originalText.trim(),
                hasHazard ? hazType : ""
            });
        }
        // Color rows: red for hazard in original, green for moved in optimized
        tbl.setDefaultRenderer(Object.class, new DefaultTableCellRenderer() {
            public Component getTableCellRendererComponent(
                    JTable t, Object val, boolean sel, boolean foc,
                    int row, int col) {
                Component c = super.getTableCellRendererComponent(
                    t, val, sel, foc, row, col);
                String haz = (String) m.getValueAt(row, 2);
                if (isOriginal && !haz.isEmpty())
                    c.setBackground(new Color(255,220,220));
                else if (!isOriginal &&
                    instrs.get(row).index != row)
                    c.setBackground(new Color(220,255,220));
                else
                    c.setBackground(Color.WHITE);
                return c;
            }
        });
    }
    // Required by AbstractMarsToolAndApplication
    protected void addAsObserver() {}
    protected void processMIPSUpdate(
        Observable o, AccessNotice n) {}
}
